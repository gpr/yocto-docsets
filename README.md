# README

Generate set of Yocto ([Dash](https://kapeli.com/dash) docsets.

## Pre-requisites

Install [pyenv](https://github.com/pyenv/pyenv) (see its documentation for more details).


## Generating all docsets

```shell
make all
```


## Available docsets

 * `bitbake.docset`: Bitbake user manual
